# 从零搭建react项目

## 环境搭建依赖

- react
- react-router4.x
- webpack
- mobx
- typescript
- sass
- axios
- jest
- enzyme

### 详细步骤

#### 创建一个空目录

- 执行mkdir reactDemo
- cd reactDemo

#### 初始化项目

- 执行npm init

提示的过程中可以一直回车，当然在作者部分可以写上您的名字

#### 安装项目必须的依赖

- yarn add react react-dom react-router typescript mobx axios

#### 安装配置weboack

##### 安装
